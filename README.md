# Milanote User Guide

A practical introduction to Milanote for creative professionals who want to organize ideas visually and collaborate effectively.

## What this guide covers
- Creating a Milanote account
- Exploring the dashboard
- Creating and organizing boards
- Adding notes, images, and links
- Sharing and collaboration
- Exporting and presenting boards

## Audience
- Designers and creatives
- Content creators
- Students and teams
- Technical writing portfolio reviewers

## Tools used
- Milanote
- Microsoft Word
- ShareX / Snipping Tool
- GitHub (documentation hosting)
